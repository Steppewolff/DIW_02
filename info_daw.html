Contenido módulos

DAW

El módulo "Despliegue de Aplicaciones Web" (DAW) trata –básicamente– de enseñar cómo desplegar, mantener y actualizar aplicaciones web en servidores.

Contenidos generales del módulo - Temario

  • Introducción a las arquitecturas web.
  • Administración de servidores web, aplicaciones y transferencia de archivos.
  • Servicios de red.
  • Sistemas de control de versiones.
  • Documentación.

DIW

El módulo "Diseño de Interfaces Web" (DIW) está enfocado –en gran medida– a aprender a integrar contenidos multimedia en aplicaciones web.

Contenidos generales del módulo - Temario

  • Interfaces gráficas.
  • Hojas de estilo.
  • Contenido multimedia e interactivo.
  • Accesibilidad web.
  • Usabilidad web.

DWES

El módulo "Desarrollo Web en Entorno Servidor" (DWES) se fundamenta en enseñar al alumno a crear –desarrollar– aplicaciones y servicios que se ejecuten por servidores en entornos web.

Contenidos generales del módulo - Temario

  • Herramientas y lenguajes de programación.
  • Uso de objetos.
  • Funciones.
  • Arrays.
  • Eventos y formularios.
  • Uso del modelo de objetos del documento (DOM).
  • Mecanismos de comunicación asíncrona.

DWEC

El módulo "Desarrollo Web en Entorno Cliente" (DWEC) tiene como fin principal que el alumno sea capaz de implementar aplicaciones que se ejecuten en navegadores web.

Contenidos generales del módulo - Temario

  • Herramientas de programacion y lenguajes en entorno servidor.
  • Inserción de código (ASP, JSP, PHP...) en páginas web.
  • Lenguajes de marcas con código embebido.
  • Generación dinámica de páginas web.
  • Acceso a datos.
  • Implementación de servicios web.
  • Programación de aplicaciones web híbridas.

EIE

El módulo "Empresa e Iniciativa Emprendedora" (EIE) pretende formar al alumno para que desarrolle su propia iniciativa profesionalmente, ya sea trabajando por cuenta ajena o como autónomo.

Contenidos generales del módulo - Temario

  • Iniciativa y creatividad emprendedora.
  • Análisis de la Pequeña Y Mediana Empresa (PYME).
  • Planificación para la creación de una empresa.
  • Introducción a la contabilidad.
  • Gestión administrativa de las empresas.

DI

El módulo "Desarrollo de Interfaces" (DI) tiene el propósito de enseñar a desarrollar interfaces de usuario, crear informes, diseñar elementos de ayuda y evaluar el funcionamiento de aplicaciones.

Contenidos generales del módulo - Temario

  • Diseño de interfaces de usuario.
  • Creación de interfaces basados en documentos XML.
  • Componentes visuales.
  • Introducción a la usabilidad.
  • Elaboración de informes.
  • Creación de documentación de aplicaciones.
  • Realización de diferentes pruebas.

PMDM

El módulo "Programación Multimedia y Dispositivos Móviles" (PMDM) está enfocado –básicamente– al desarrollo de software (apps multimedia y juegos) para los dispositivos móviles.

Contenidos generales del módulo - Temario

  • Análisis de tecnologías para el desarrollo de aplicaciones en diferentes dispositivos móviles.
  • Programación de aplicaciones multiplataforma.
  • Librerías multimedia.
  • Uso de motores de juegos.
  • Programación de juegos 2D y 3D.
